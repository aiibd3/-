# -
```
/
├── bin      - Essential binaries for users and system
├── boot     - Files needed to boot the system (e.g., kernel, bootloader)
├── dev      - Device files (e.g., hardware and virtual devices)
├── etc      - System configuration files
├── home     - User home directories (e.g., /home/username)
├── lib      - Shared libraries essential for binaries in /bin and /sbin
├── media    - Mount points for removable media (e.g., USB drives, CDs)
├── mnt      - Temporary mount points for filesystems
├── opt      - Optional software packages
├── proc     - Virtual filesystem for process information
├── root     - Home directory for the root user
├── run      - Runtime data for processes
├── sbin     - System binaries for administrative tasks
├── srv      - Data for services provided by the system
├── sys      - Virtual filesystem for system information
├── tmp      - Temporary files (cleared on reboot)
├── usr      - Secondary hierarchy for user programs and data
│   ├── bin   - User binaries
│   ├── lib   - Libraries for /usr/bin and /usr/sbin
│   ├── sbin  - System binaries for users
│   └── share - Shared data (e.g., icons, documentation)
├── var      - Variable data (e.g., logs, spool files, databases)

```
